# PokéMaster Online

PokéMaster Online é um servidor de PokéTibia baseado em TFS 1.4 com OTCv8.  
Este repositório contém os arquivos do client (ou servidor), usados para o desenvolvimento e personalização da interface, HUD e sistemas.

## Estrutura

- `/modules/`: Módulos do OTCv8, incluindo login, HUD, inventário e mais.
- `/mods/`: Arquivos de interface visual chamados OTUI no geral.
- `/data/`: Scripts do servidor (em caso de incluir a parte server-side).

## Objetivo

Este projeto visa modernizar completamente o visual e funcionamento do client OTCv8 com suporte para mobile, além de implementar sistemas avançados de gameplay e interface.
